# Builder Project Template

Builders interested in the **LAB Open Source Builders Fund** can apply through the official LAB builder application form.

Developers who prefer submitting through GitHub may open a project proposal using this template.

Please note that Selected projects will be asked to submit their project on the LAB Artizen Fund Page to participate in the matching pool.

Projects are reviewed by the LAB team based on:

- technical progress
- ecosystem impact
- open-source contribution
- relevance to Bitcoin infrastructure
